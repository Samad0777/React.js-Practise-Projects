# Todo App ✅

A simple and responsive Todo application built with **React.js** and **Tailwind CSS**.  

## Features
- ➕ Add new todos  
- ✔️ Mark todos as completed  
- 🗑️ Delete todos  
- 💾 Data is saved in **localStorage**, so it stays even after refreshing the page  
- 📱 Fully responsive design  

## Tech Stack
- React.js ⚛️  
- Tailwind CSS 🎨  
- LocalStorage 💾
